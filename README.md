# AeroFit Treadmill Customer Distribution
A Tableau dashboard project analysing customer purchase behaviours for the treadmill models of AeroFit.
## Access to Live Dashboard 
https://public.tableau.com/app/profile/jiyansh.garg/viz/AerofitTreadmillCustomerDistribution_17633699202130/Dashboard1

## Project Overview
This project leverages the customer purchase dataset of AeroFit treadmill models (KP281, KP481, KP781) to understand which customer segments buy which model and why. Using a combination of descriptive statistics, visualisation and probability/contingency-table analysis, the goal is to provide actionable insights for marketing, product positioning and customer targeting.

The interactive dashboard built in Tableau allows stakeholders to explore customer distributions across age, gender, education, income, fitness level, treadmill usage and miles expected per week, in relation to the model purchased.
